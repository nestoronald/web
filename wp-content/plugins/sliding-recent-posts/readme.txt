Displays a sliding widget on your site that includes your most recent posts.
== Description ==

This plugin displays a widget on your site that will slide into the screen when you click on it and it will display a list of recent posts with thumbnails

== Installation ==

1. Upload 'sliding-recent-posts' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on Recent Posts in the WordPress Dashboard sidebar
4. Set the options you desire and save
