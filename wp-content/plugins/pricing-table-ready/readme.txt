Pricing Table Ready! WordPress plugin allow you generate and manage CSS pricing table or comparison table with table generator in the easy way.

== Description ==

With [WordPress Pricing Table plugin](http://readyshoppingcart.com/product/pricing-table-ready-plugin-pro/ "WordPress Pricing Table plugin") you can manage pricing table, publish pricing table on WordPress page or post. 
Pricing Table Ready! plugin include five different styles so you can choose style for your online store or blog.
Create Responsive pricing table, comparison table is also available to creation.

This version **dont compatible** with 0.2.9.2 version. you must to create all pricing tables one more time. If you want to use old version, download it here: http://readyshoppingcart.com/downloads/pricingtables/wp-plugin-pricing-table-ready.zip

To insert pricing table use shortocode:

* [ready-pricing-tables id=123 template=tiny]
* **id** price template ID
* **template** template design name

Features:

* Five different table templates
* Pricing Table live preview
* Easy to use price table generator
* Drag and drop columns
* Unlimited columns and rows
* Button, text, special style for any pricing gird
* Tooltips and icons for pricing table cells
* Tonns of amazing pricing tables in PRO version

== Installation ==

Easy installation

Go to the plugin section of your WordPress installation and search for "Pricing Table Ready!" in the search browser. Install the plugin and activate it.

Manual installation

1. Download and extract the zip file and upload it to the /wp-content/plugins/ directory.
2. Activate the Pricing Table Ready! plugin through the "Plugins" menu in WordPress.

Further steps

1. Create a table, and add rows
2. Choose pricing or comparison table style
3. Fill table with data and save it
4. Add table shortcode to a page or post
